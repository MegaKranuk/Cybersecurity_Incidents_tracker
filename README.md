# Трекер інцидентів кібербезпеки

Вебзастосунок для реєстрації, моніторингу та управління кіберінцидентами.
Робота виконана в межах лабораторної роботи №1 (Варіант 2).

## Реалізований функціонал
- **Створення:** введення даних про інцидент з валідацією;
- **Редагування:** можливість зміни вже створених записів;
- **Видалення:** видалення неактуальних інцидентів;
- **Сортування та фільтрація:** сортування за датою, фільтр за типом та критичністю;
- **Збереження:** дані автоматично зберігаються у **LocalStorage** браузера;
- **Відображення:** динамічний рендер таблиці.

---

##  Валідація введених даних
- дата інциденту — обовʼязкова, не пізніше поточної;
- тип інциденту — обовʼязкове поле;
- критичність — обовʼязкове поле;
- імʼя репортера — від 5 до 15 символів;
- опис інциденту — від 15 до 50 символів.

---

## Використані технології
- **HTML5:** семантична розмітка;
- **CSS3:** Flexbox, змінні, адаптивність;
- **JavaScript:** DOM API, Event Delegation, LocalStorage, JSON.

---

## Запуск проєкту
1. Склонуйте репозиторій.
2. Відкрийте файл `index.html` у браузері.
3. Додаткові залежності відсутні.
